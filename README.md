# SJCIT-CAMPUS-NAVIGATION

An interactive web-based campus navigation system developed for **SJC Institute of Technology (SJCIT)**. The application helps students and visitors easily locate campus buildings and provides route guidance from the main entrance to selected destinations.

---

## 📌 Features

- 🎓 Welcome screen with user name and email validation
- 🛰️ Interactive satellite campus map
- 📍 View all major campus buildings
- 🛣️ Route navigation from the entrance to any selected block
- 📖 Information panel for every department/building
- 📱 Responsive user interface
- 🎯 Smooth map animations
- 🗂️ Sidebar navigation menu
- 🔄 Recenter and Follow controls
- ⚡ Straight-line fallback if routing service is unavailable

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Leaflet.js
- Leaflet Routing Machine
- OpenStreetMap Routing Service (OSRM)
- Google Satellite Tiles
